# The InfiniteLimits Website

This is the source code for <https://inflmts.com>.

## Setup

Install dependencies:

```
pnpm install
```

## Develop

```
pnpm run dev
```

## Build

```
pnpm run build
```

## Publish

[@inflmts/gh-pages](https://npmjs.com/package/@inflmts/gh-pages) is required.

```
pnpm add -g @inflmts/gh-pages
pnpm run publish
```
